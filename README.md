# Calculadora de Variables

Un script simple en Python que demuestra operaciones aritméticas básicas utilizando variables numéricas.

## Descripción

Este proyecto contiene un archivo Python que:
- Define múltiples variables numéricas
- Realiza sumas entre ellas
- Muestra el resultado por consola

## Requisitos

- Python 3.x

## Uso

Ejecuta el script desde la terminal:

```bash
python archivo.py
```

## Ejemplo de Salida

```
24
```
